Instagram Captions Generator

Role:

  • Project manager
  • Technical realization

Problem:

For the 3rd Semester Web API Development elective me and my group were tasked to create a project, that will show our knowledge of working with databases and API's. The topic of the project was a free theme.

Solution:

Solution consists of a Instagram Captions Generator, that sends a call to the database and gets a caption of desired length and mood.

Project management:

As a project manager, my main tasks were:

  • Organization of weekly meetings
  • Quality control
  • Ensuring that the project is progressing within the scope of the assignment and the clients expectations
  • Organization of time schedule and the task board

Technical realization:

The website is realized though HTML, CSS and Javascript, as well as MSSQL database was used for storing the captions. Github was used for version control.
Users can login to the website in order to make new captions and send them for approval by the admins. This was done through routing. The approved creators are added to the database and get assigned with a role. Admins can add the captions and approve them as well as approve other admins. Creators can submit captions and see what captions they have submitted. All the possible endpoint are connected with the GET, POST, PUT and DELETE methods (in RESTful API conventions).
